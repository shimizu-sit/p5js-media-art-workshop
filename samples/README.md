# Samples

このフォルダには，p5.jsでメディアアートを作るためのサンプルコードがあります．

各フォルダの `sketch.js` をコピーして，p5.js Web Editorに貼り付けると実行できます．

p5.js Web Editor：


https://editor.p5js.org/

## 基本的な使い方

1. 試したいサンプルのフォルダを開く
2. `sketch.js` を開く
3. コードをすべてコピーする
4. p5.js Web Editorを開く
5. コードを貼り付ける
6. ▶ボタンを押す

## 改造のコツ

まずは，次のような数字を変えてみましょう．

* `background()` の数字
* `fill()` の数字
* `circle()` の数字
* `speed` の数字

プログラムは，少し変えるだけで見た目が大きく変わります．
