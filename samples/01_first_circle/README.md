# 01 first_circle

## 内容

画面の中央に円を描く，最初のサンプルです．

p5.jsで図形を描く基本を確認します．

## 学べること

- `createCanvas()` で画面を作る
- `background()` で背景色を決める
- `fill()` で図形の色を決める
- `circle()` で円を描く

## 使い方

1. `sketch.js` の中身をコピーする
2. p5.js Web Editorを開く
3. 左側のコードをすべて消す
4. コピーしたコードを貼り付ける
5. ▶ボタンを押して実行する

p5.js Web Editor：

https://editor.p5js.org/

## 改造ポイント

次の部分の数字を変えてみましょう．

```javascript
background(240);
fill(255, 120, 120);
circle(200, 200, 120);
```

|    変更する場所    | 変わるもの |
| ------------------ | ---------- |
| `background()`     | 背景色     |
| `fill()`           | 円の色     |
| `circle()` の1つ目 | 円の横位置 |
| `circle()` の2つ目 | 円の縦位置 |
| `circle()` の3つ目 | 円の大きさ |

## チャレンジ

* 円の色を好きな色に変える
* 円を大きくする
* 円の位置を左上や右下に移動する
* 背景色を黒にしてみる
